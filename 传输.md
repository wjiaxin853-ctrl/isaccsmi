## isaccsmi 使用说明

这个项目主要是 `Isaac Sim` 的脚本工作目录，用来打开场景、调整火车物理属性、设置速度和导出数据。

### 1. 启动 Isaac Sim

```bash
export LD_LIBRARY_PATH=/usr/local/cuda-12.1/targets/x86_64-linux/lib
cd /home/admin204/isaac
./isaac_sim.sh
```

如果在 Linux 上使用，先把 `ISAAC_ROOT` 改成你的 Isaac Sim 安装目录，再执行同样的启动脚本。

### 2. 常用脚本

```bash
python3 调节火车速度.py
python3 给火车物理属性让火车行走.py
python3 增加环境.py
python3 查看Stage内容的脚本.py
```

### 3. Principia 源码安装

如果你已经把 `principia-cli` 源码放到本地，可以按下面的方式安装：

```bash
cd /Users/wjx/Documents/wjx/principia-cli
npm config set registry https://registry.npmmirror.com/
rm -rf node_modules
ONNXRUNTIME_NODE_INSTALL_CUDA=skip npm install
#npm install
npm run cli:build
cd cli && npm run link
principia --help
```

### 4. Principia 常用命令

```bash
principia auth
principia history
principia -T <taskId>
principia "分析这个 Isaac Sim 项目"
```

### 5. 当前项目的火车场景

- 火车主模型主要在 `/World/Obj3d66_1238088_1_934`
- 另一个火车节点在 `/World/AM223_04`
- 直线运行一般是直接写 `physics:velocity`
- 如果要先看场景结构，先跑 `查看Stage内容的脚本.py`
